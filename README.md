How to use
Run in your IDE, other programs, or CMD (depending on what you're using).

In IDE: type in the terminal
In CMD or other programs: just type directly

text
go run main.go
Then:

Enter amount in RUB

Enter currency code (USD, EUR, CNY, GBP, TRY)

Get converted amount

Type y to convert again or n to exit

Example
text
1 USD = 95.50 RUB
1 EUR = 103.20 RUB
1 CNY = 13.40 RUB

Enter RUB amount: 1000
Convert to (USD, EUR, CNY, GBP, TRY): USD
= 10.47 USD
Again? (y/n): n
